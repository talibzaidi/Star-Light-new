2.2 Modifying Products

  The Product page has three functions.

1) Choose a product and click on edit to edit an existing product.

2) Fill in the form to create a new Product. ( see below )

3) Choose a product and click on delete to permanently remove a product from the catalogue.

CREATING PRODUCTS:

PRODUCT NAME: [ 250 Characters max. ] Enter the name of the product that you are creating. An example of this would be ' Namcorp's - Digital Phone - 8810x '. Do not precede the name with spaces or special characters ( $%^&*etc.) This is the name that people see for a product and ideally should be short and to the point. Use Areas and Sub-areas to organize your site!

SUB AREA SELECT: Now that your product has a name put it into a sub area. If for some reason the sub area does not exist at this time, you can edit the product later.

PRODUCT ID: Every product needs a unique numeric product id. You cannot enter a product if you do not give it a numeric id, and only numbers are acceptable in this field. If you do not have numeric product id's at present now would be the time start assigning them. Numbers can range from 1 to over 16 digits. Preceding zero's are stripped from the number and will not be displayed. ( 000034 == 34 )

UPC CODE: [ 250 chars max. ] For your own internal reference.

SERIAL NUMBER: [ 250 chars max ] For your own internal reference.

DESCRIPTION: [ 250 chars max ]. This is used as a short information field. Ideally if the PRODUCT NAME is a model number you would put the name of the manufacturer here. This field is visible on both the catalogue pages and the detail pages.

TEXT 1: [ 65000 chars max. (1k max. recommended) ] Accepts HTML as input ( do not include <html> <body> <head> tags all other tags acceptable ). Inserting HTML or scripts into these fields may produce unexpected behaviour and is not recommended for inexperienced users. Small errors may be magnified. Field is used to describe product and is not required.

TEXT 2: [ 65000 chars max. (1k max. recommended) use ~ to bullet] This text area applies to all the same rules as the above text area but appears on the detail page as a bulleted list. Use the tilde key SHIFT+` to get ~ to insert a bullet point.

THUMBNAIL: [ 100xN pix. ] Image used in specials and catalogue. Can be any reasonable height ( i.e. 100x150pix vs. 100x2678pix ). Must be 100 wide or will be formatted thus and may distort. Enter only the image name: i.e. xlr700-4t.gif the 't' in this example is used to signify thumbnail and is not required but is recommended. See uploading Images for more info.

IMAGE: [ 325xN pix. ] Image used for detail page can be up to 325 pixels wide but can be smaller. Images larger than 325 pixels wide will produce undesired behaviour with the site graphics. Enter only the image name: i.e. xlr700-4.gif

MANUFACTURER'S LINK: [ 250 char max.] Optional link to the manufacturer to appear on the detail page. Must be a valid web-site address i.e. http://, ftp://, etc.

COST: [ double ] Product cost used in calculations.

MSL: [ double ] Suggested list price used in calculations.

WEIGHT: [double ] Unused.

SPECIAL: [ boolean ] Items checked will randomly appear throughout the site as a special.

NEW PRODUCT: [ boolean ] Items checked will appear in the site as a new product.

OVERSIZE: [ boolean ] Unused.
