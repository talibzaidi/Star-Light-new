2.1.1 Sub Area Creation

  Sub Area Creation is very similar to Area creation, there is only one additional step. Type the Name of the Sub Area into the text box to create a new Sub Area, then select an area from the Area Box to associate that Sub-Area with an Area. Finally click the submit button. To delete a sub-area simply pick the sub-area from the lowest drop-down box and click on delete. As with areas, you can return to this page from the main menu to correct ( delete and re-enter ) or to add new sub areas to an existing area. We recommend having no more than 250 sub-areas per area.
