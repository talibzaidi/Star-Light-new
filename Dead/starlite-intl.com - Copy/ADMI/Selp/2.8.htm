2.8 Making Links

Links are managed in the following ways:

1) Edit an existing link by ID and Name

2) Entering a new link.

LINK NAME: [ 250 char max. ] Choose a name for the link that will tell you ( or others ) whose link this is or what it does. This allows others and yourself to find a link later on to delete or edit it.

LINK URL: [ 250 char max. ] Any valid address on the internet.

LINK TITLE: [ 250 char max. ] What your users will see to describe this link. i.e. Morningstar Laboratories

IMAGE: An image that you have uploaded via the Image Upload Page. Follows the same rules as other images.

3) Delete an existing link by ID and Name
