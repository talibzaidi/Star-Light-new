2.1 Creating Areas

  The Area Create page is fairly easy to use. There only two fields that can be used. To create a New Area all you have to do is type the name of the new area into the text field and click on the button that is labeled 'Submit'. To delete an area choose the area you wish to delete from the drop-down box and click the 'Delete' button. If you make a mistake all you have to do is return to this page from the Sanction homepage and delete the Area that is mispelled. This page will send you to the Sub-Area Create page, after submission.
